Welcome to the AutoCongrats App blog. See what the App offer to you!

AutoCongrats helps you to congrats your contacts!

You don’t need to worry about if you don’t remember some friends name day or birthday.

Application checks from your contacts if any persons is having a name day or birthday.

Congratulate your friends in a care free way.

The App show to you who’s name day or birthday for day go on.

Just click tulip flower icon if you want congrats. You decided if you want to congrats.

Setting up preferences

Allowing push notifications, you will receive a daily push notification as reminder on chosen time.

Choosing language affects how application can find persons having a name day. Also applications language is changed based on that.

Set default content of the congratulate message, however you can modify message before actually sending it.

Use XXXX characters to place where you want person’s name.

Set WhatsApp as your primary message channel. If your friend doesn’t have WhatsApp then message will be sent as SMS.

Use it as you want

Maybe you just want to see what names have name days. Click on the red text TODAY NAMEDAYS
